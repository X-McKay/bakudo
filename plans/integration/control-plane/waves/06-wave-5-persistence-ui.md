# Wave 5: Persistence & UI

**Goal:** Store the sandbox lifecycle state (preserved, merged, discarded) in the session record, update the approval copy to explain worktree states instead of just exit codes, and update the inspect tabs to show the new host artifacts (`patch.diff`, `changed-files.json`).

**Non-Goals:** Do not build the new "Candidate Set" UI yet (that's UX Realignment). This wave just makes the data available and fixes the text of the existing approval prompt.

## Pre-reads & Vocabulary
- Review `2026-04-19-bakudo-abox-control-plane-spec.md` (Section: Phase 3 - State Tracking & Interactive Merge).
- **SandboxLifecycleState:** An enum representing the current state of a sandbox (`ephemeral`, `preserved_active`, `preserved_merged`, `preserved_discarded`).

### 2. Update `sessionController.ts`

When an attempt finishes, record its lifecycle state.

```typescript
// src/host/sessionController.ts
// After executeAttempt returns:
let lifecycleState: SandboxLifecycleState = "ephemeral";

if (plan.profile.sandboxLifecycle === "preserved") {
  if (executionResult.status === "succeeded" && plan.profile.mergeStrategy === "auto") {
    lifecycleState = "preserved_merged";
  } else if (executionResult.status === "blocked") {
    lifecycleState = "preserved_active";
  } else {
    lifecycleState = "preserved_discarded";
  }
}

const attemptRecord: SessionAttemptRecord = {
  attemptId: plan.spec.attemptId,
  status: executionResult.status,
  dispatchPlan: plan,
  sandboxLifecycleState: lifecycleState,
};
```

*Note: You will also need to add a new function `resolveBlockedAttempt(attemptId, action)` that updates the state to `preserved_merged` or `preserved_discarded` when the user responds to the approval prompt.*

### 3. Update Approval Copy (`approvalPromptCopy.ts`)

Change the copy to reflect worktree states.

```typescript
// src/host/renderers/approvalPromptCopy.ts
export const renderApprovalPrompt = (attempt: SessionAttemptRecord): string => {
  if (attempt.sandboxLifecycleState === "preserved_active") {
    return "Agent completed its work and the sandbox is preserved.\nReview the changes using `/inspect` and then choose an action:\n\n[y] Merge changes\n[n] Discard sandbox";
  }
  return "Task finished. Approve outcome?"; // Fallback
};
```

### 4. Update Inspect Tabs (`inspectTabs.ts`)

Add the host artifacts to the artifacts tab.

```typescript
// src/host/inspectTabs.ts (in Artifacts tab formatter)
const artifacts = attempt.executionResult?.artifacts || [];
if (artifacts.includes("patch.diff")) {
  output.push("Host Artifacts:");
  output.push("  - patch.diff (Generated by host inspection)");
  output.push("  - changed-files.json (Generated by host inspection)");
}
```
